# Word Game v1.0 🎯

一个单词猜测游戏，类似Wordle，支持中英双语词汇学习。

## 🌟 游戏特色

### 🎮 核心功能
- **经典Wordle体验**：6次机会猜中7字母单词
- **智能颜色提示**：绿色(正确位置)、黄色(存在但位置错)、灰色(不存在)
- **双语词汇支持**：六级英语词汇 + 雅思学术词汇
- **智能键盘禁用**：不存在于目标单词的字母自动禁用，防止无效输入
- **双人对战模式**：实时同步，两台设备做同一题，看谁更快猜中！

### ⌨️ 多种输入方式
- **物理键盘**：支持字母输入、Backspace删除、Enter提交、Ctrl+N新游戏
- **虚拟键盘**：触摸友好的虚拟键盘，支持所有操作
- **智能焦点管理**：输入光标自动移动，删除时智能定位

### 📱 完美适配
- **响应式设计**：桌面、平板、手机全适配
- **流畅体验**：无卡顿、无跳动、快速响应
- **无障碍操作**：支持键盘导航和屏幕阅读器

## 🚀 快速开始

### 本地运行
```bash
# 启动本地服务器
python3 -m http.server 8000

# 访问游戏
# 打开浏览器访问: http://localhost:8000
```

### 配置 Firebase（双人对战功能需要）

1. 创建 Firebase 项目（参考 `FIREBASE_SETUP.md`）
2. 在 `script.js` 中更新 Firebase 配置信息
3. 启用 Realtime Database

详细步骤请查看 [FIREBASE_SETUP.md](./FIREBASE_SETUP.md)

### 游戏玩法

#### 单人模式
1. **开始游戏**：点击"NEW"按钮开始新回合
2. **输入猜测**：使用键盘或虚拟键盘输入7字母单词
3. **查看提示**：根据颜色提示调整下次猜测
4. **获胜条件**：在6次机会内猜中正确单词

#### 双人对战模式
1. **创建房间**：点击"双人"按钮，然后点击"创建房间"
2. **分享房间ID**：复制6位房间ID，发送给对手
3. **加入房间**：对手输入房间ID，点击"加入房间"
4. **开始对战**：双方看到相同的目标单词，实时看到对方进度
5. **决出胜负**：谁先猜中或使用更少次数获胜

## 🎯 智能功能

### 🧠 键盘智能禁用
- **自动学习**：当字母被确认为不存在于目标单词中时，自动禁用对应键盘按钮
- **双向同步**：物理键盘和虚拟键盘状态实时同步
- **视觉反馈**：禁用的按钮显示为半透明状态，防止误操作
- **游戏重置**：新游戏开始时所有按钮重新启用

### 🎯 颜色提示系统
- 🟢 **绿色**：字母正确，且位置正确
- 🟡 **黄色**：字母存在于目标单词中，但位置错误
- ⚫ **灰色**：字母不存在于目标单词中

## 🎯 游戏规则

### 基本规则
- 目标：猜中系统随机选择的7字母单词
- 机会：共有6次猜测机会
- 输入：每次必须输入完整的7字母单词

### 颜色提示系统
- 🟢 **绿色**：字母正确，且位置正确
- 🟡 **黄色**：字母存在于目标单词中，但位置错误
- ⚫ **灰色**：字母不存在于目标单词中

### 获胜条件
- ✅ **胜利**：在6次机会内猜中正确单词
- ❌ **失败**：6次机会用完仍未猜中

## 🎮 双人对战功能

### 功能特点
- **实时同步**：使用 Firebase Realtime Database 实现实时数据同步
- **房间系统**：6位随机房间ID，简单易用
- **进度显示**：实时查看对手的猜测进度和结果
- **状态同步**：自动同步游戏状态，无需手动刷新

### 使用流程
1. **玩家A**：点击"双人" → "创建房间" → 复制房间ID
2. **玩家B**：点击"双人" → 输入房间ID → "加入房间"
3. **开始游戏**：双方自动开始，看到相同的目标单词
4. **实时对战**：每次猜测后，对方立即看到你的进度
5. **查看结果**：游戏结束后，可以看到双方的完成情况

### 注意事项
- 需要配置 Firebase 才能使用双人对战功能
- 建议在同一网络环境下使用，延迟更低
- 房间在30分钟无活动后会自动清理（可在 Firebase 控制台配置）
